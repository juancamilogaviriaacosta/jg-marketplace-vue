# 🛒 Juan Gaviria Marketplace

Aplicación web de **tienda virtual (Marketplace)** desarrollada como MVP utilizando **Vue.js + Spring Boot**.  
Permite gestionar productos, usuarios y realizar el flujo completo de compra.

---

## 🚀 Tecnologías

### 🖥️ Frontend
- Vue 3
- Vite
- TypeScript
- HTML5
- CSS3

### ⚙️ Backend
- Java 21
- Spring Boot
- Spring Data JPA

### 🗄️ Base de Datos
- PostgreSQL

---

## 📦 Funcionalidades

- 👤 Gestión de usuarios  
- 📦 Gestión de productos  
- 🛒 Carrito de compras  
- 💳 Flujo básico de compra  

---

## 🐳 Ejecución con Docker

Puedes levantar todo el proyecto fácilmente con:

```bash
docker compose up --build
```

## Abrir el navegador

- http://localhost:8080
